spyd3rwebs runs on a custom built Python automation stack. A master aggregator pulls from 50+ RSS sources across dozens of topic categories (breaking news, finance, politics, health, tech, food, travel, and more). The aggregator grabs the 5 most recent stories from each source and shuffles them to ensure a different source always rises to the top. Every run outputs the results as XML feeds. For sites that don't publish RSS feeds, custom scraper scripts build them. Before anything goes live, a separate AI rewriter rewrites every headline and summary in its own words, preserving all links, dates, and structure, so only original content gets published. The newsletters you see here are built entirely from those feeds.

RSS Truncater v1.1

A command-line utility that scans a folder for RSS / XML feed files and trims each one down to a maximum number of items as desginated by user. (Keeping the most X number of recent entries and discarding the rest.) Supports RSS 2.0 and Atom feeds across macOS, Windows, and Linux. (Windows and Linux not tested). Free to use, not for resale.
